# Discord Content Bot

بوت ديسكورد بيولّد بوستات جاهزة لتويتر (نص + صورة اختياري)، وبيقدر يجيب ترندات
حقيقية أوتوماتيك. ما بينشر أي شي على تويتر — انتي يلي بتنسخي وتلصقي يدوياً.

## 1. سوّي بوت ديسكورد (مجاني)

1. روحي على discord.com/developers/applications
2. New Application → اسم البوت
3. تبويب Bot → Reset Token → انسخي التوكن (هاد `DISCORD_BOT_TOKEN`)
4. تبويب OAuth2 → URL Generator → اختاري scopes: `bot`, `applications.commands`
   → بالصلاحيات اختاري Send Messages, Embed Links → انسخي الرابط وافتحيه لتضيفي
   البوت لسيرفرك

## 2. مفتاح Gemini (مجاني)

من aistudio.google.com/apikey — اعملي مفتاح مجاني (فيه حد استخدام يومي مجاني
كافي لهيك استخدام).

## 3. مفتاح Pexels (اختياري - لميزة الصور)

من pexels.com/api — مجاني، وبيدّيكي صور حرة الترخيص بالكامل (مسموح تنشريها
بدون مشاكل حقوق نشر). لو ما حطيتي المفتاح، ميزة `with_image` رح تقول ما لقيت
صورة بس باقي الأوامر بتشتغل عادي.

## 3. التثبيت

```bash
python3 -m venv venv
source venv/bin/activate   # على ويندوز: venv\Scripts\activate
pip install -r requirements.txt
cp .env.example .env
# افتحي .env وحطي DISCORD_BOT_TOKEN و GEMINI_API_KEY
python bot.py
```

## 4. كيف تستخدميه

اكتبي `/post` بس (أو `/post trend:كذا` لموضوع محدد)، والبوت بيطلعلك:

- 3 خيارات بوست بأزرار **① ② ③** — دزّي على اللي بيعجبك
- زر **🔄 جدّدي الخيارات** — يطلعلك 3 خيارات جديدة نفس الترند
- زر **🎲 ترند جديد كلياً** — يبدل الترند بالكامل

بعد ما تختاري بوست، بيطلعلك زرين:
- **🖼️ ضيفي صورة** — صورة حرة الترخيص مناسبة للموضوع
- **🎥 ضيفي فيديو** — فيديو قصير حر الترخيص مناسب للموضوع

- `/trends` — تعرضلك أهم 8 عناوين أخبار/ترندات حالية بدون توليد
- `/reply original:النص` — رد جاهز على تغريدة معينة (تستخدميها يدوياً بحذر،
  رد مدروس مش رد جماعي آلي)

## ملاحظات مهمة

- **الترندات**: مصدرها Google News RSS (مجاني، حقيقي، بالعربي حسب الأردن).
  مش ترندات تويتر بالذات (X ألغت الوصول المجاني لهاي)، بس أقرب بديل حقيقي
  ومجاني موجود.
- **الصور والفيديوهات**: من Pexels (مجاني، بترخيص حر بالكامل — مسموح تنشريهم
  بدون مشاكل حقوق نشر). ملاحظة: مش صور/فيديوهات "الخبر نفسه" لأن هاي محمية
  بحقوق نشر، بس صور/فيديوهات مناسبة لموود وموضوع البوست.
- **كل شي هون توليد بس، النشر يدوي بالكامل** — ولا مفتاح X ولا صلاحية نشر
  بالأداة خالص.
